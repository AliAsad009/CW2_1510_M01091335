
## IT Ticket & Incident Management Dashboard

This project is a Streamlit-based web application for managing IT tickets, cyber incidents, and user accounts. It features interactive dashboards, data visualization, and AI-powered chat support using OpenAI. The system allows users to view, create, and update tickets and incidents, and provides secure authentication and user management.

**Key Features:**
- Streamlit web interface for easy interaction
- Ticket and incident tracking
- User authentication and management
- Data visualization with Altair and Plotly
- AI chat integration for support
